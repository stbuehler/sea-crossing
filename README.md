A Conhex-like game.

Dependencies:
 * A C++11 compiler.
 * [glfw >= 3](http://www.glfw.org/)
 * [GLEW](http://glew.sourceforge.net/ "The OpenGL Extension Wrangler Library")
 * [GLM](http://glm.g-truc.net/ "OpenGL Mathematics")
 * OpenGL

This game is aimed to run on Linux and Windows machines.
